Technologies Used:
- Python 3
- Regular Expressions (re library)
- Data Cleaning and Text Processing
- Conditional Analysis (Rule-based scoring)
- Command Line Interface (CLI)
- Pydroid 3 / Jupyter Notebook (optional)

Project Description:
This project is a Password Strength Checker built using Python. It evaluates the strength of a password based on key security factors such as length, use of uppercase and lowercase letters, numbers, and special characters. Based on the analysis, it classifies the password as Weak, Medium, or Strong and provides suggestions to improve password security.
